# Stage 01 Project

## Description

This is stage 01 web app api project that uses Golang, Gin, and PostgreSQL. This project implement clean architechture with JWT authentication.

## Features

- Authentication
- Admin page
- List of news
- Create news
- Edit news
- Filter news
- Purchase subscription
